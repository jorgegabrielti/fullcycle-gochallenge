## Desafio:
Publicar uma imagem no docker hub. Quando executarmos:

```
docker run <seu-user>/fullcycle
```

Temos que ter o seguinte resultado: 
```
Full Cycle Rocks!!
```
### Requisito:
A imagem de nosso projeto Go precisa ter menos de 2MB.
